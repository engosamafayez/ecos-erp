# styles

Global styling lives in [`src/index.css`](../index.css) — the Tailwind CSS v4
entry point, theme tokens (light/dark via CSS variables), and base layer.

This folder is for **additional** global stylesheets as the app grows
(e.g. font-face declarations, print styles, vendor overrides). Component-level
styles are expressed with Tailwind utility classes co-located in each component.
