# Helpers

**Namespace:** `App\Helpers`

Stateless helper classes and (optionally autoloaded) helper functions for small,
cross-cutting utilities — formatting, math, string manipulation, etc.

**Convention:** Helper classes are `PascalCase` ending in `Helper`
(e.g. `MoneyHelper`) with `static` methods. Global function files, if any, are
registered via Composer `autoload.files`.
