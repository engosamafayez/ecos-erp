# Contracts

**Namespace:** `App\Contracts`

Application-wide interfaces (ports) that define behavior without implementation —
the seams that enable dependency inversion across the app.

**Convention:** `PascalCase`, named by capability. Use an `Interface` suffix
(e.g. `PaymentGatewayInterface`) **or** a role-based noun (e.g. `Clock`) —
pick one style and apply it consistently (see NAMING-CONVENTIONS.md).
