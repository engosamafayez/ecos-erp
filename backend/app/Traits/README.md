# Traits

**Namespace:** `App\Traits`

Reusable PHP traits providing shared, horizontal behavior (e.g. `HasUuid`,
`Cacheable`). Use sparingly — prefer composition/services when behavior carries
its own state or dependencies.

**Convention:** `PascalCase`, typically a `Has*`, `Is*`, or capability name
(e.g. `HasTimestampsInTimezone`).
